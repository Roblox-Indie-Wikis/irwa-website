# About
This is the repository for the Independent Roblox Wiki Alliance site, which you can view at https://indierobloxwikis.org. Join our [Discord server](https://dsc.gg/irwa)!

The site is built using [Jekyll](https://jekyllrb.com/).

# Contributing
To contribute, you can open a Pull Request with your changes.

# Adding your wiki

If you have been accepted into the alliance, your request should automatically be processed by a coordinator. If you don't see it within 3 days on the website, open an issue with all the details.

# Internal steps to adding wiki
Updates are now data-driven:
- Insert logo into `media/`
- Add wiki details to `_data/members.yml` (the carousel on the front page and the members list are both generated from this file).
